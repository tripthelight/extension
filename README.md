# 크롬 기반 브라우저 확장 프로그램

## CHANNEL BLOCKER
1) 로그인 되어있는 상태일 경우, 
   - tp-yt-iron-dropdown > yt-list-item-view-model 에 "채널 추천 안함", "관심 없음"이 있는 경우
   - 메인화면, 상세 추천영상에서는 "채널 추천 안함", "관심 없음" 버튼 노출되면 안됨
2) 브라우저 상단 확장프로그램 아이콘 누르고 나오는 차단 리스트들 오른쪽에 화살표 추가해서 클릭하면 전체 차단 채널 팝업 띄워야 됨
3) "채널 추천 안함", "관심 없음" 버튼 왼쪽에 SVG 아이콘 필요 
4) 메인화면에서 간혈적으로 "채널 추천 안함"이 안먹히는 영상이 있음
